## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → POST /api/export/pdf and /api/export/docx | Posted ReportViewModel re-validated by ReportViewModelSchema before any chart render/rasterize |
| chart data → SVG string → @resvg/resvg-js → PNG | Chart SVG is built server-side from validated numeric vm.hospMetrics; rasterizer consumes only app-generated SVG |
| new dependency → server bundle | @resvg/resvg-js NAPI binary loaded server-side via serverExternalPackages |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-7-04 | Tampering | SVG fed to @resvg/resvg-js | mitigate | The SVG is generated by `renderChartSvgString` from `buildChartData(group)` — values come from `vm.hospMetrics` (validated `z.number().nullable()` via HospMetricSchema); colors from the closed CHART_SERIES enum.
